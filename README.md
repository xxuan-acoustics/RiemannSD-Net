## 📋 Mathematical Foundations

Our paper aims to bridge classical mathematics with modern speech processing. The theoretical essence of this paper is drawn from the classical theories of two renowned mathematicians:

* **[Pafnuty Chebyshev](https://en.wikipedia.org/wiki/Pafnuty_Chebyshev)**: *[Chebyshev polynomials](https://en.wikipedia.org/wiki/Chebyshev_polynomials)*.
* **[Bernhard Riemann](https://en.wikipedia.org/wiki/Bernhard_Riemann)**: *[Riemannian manifolds](https://en.wikipedia.org/wiki/Riemannian_manifold)*.

## Getting Started

### 1 Setup Environment
You need to create the running environment by [Anaconda](https://www.anaconda.com/).

First, create and activate the environment:

```bash
conda create -n RiemannSD-Net python=3.11.11
conda activate RiemannSD-Net
```
Then install the requirements:[![speechbrain](https://img.shields.io/badge/speechbrain-1.0.3-FF6F00?logo=pytorch&logoColor=white)](https://pypi.org/project/speechbrain/1.0.3/)
[![librosa](https://img.shields.io/badge/librosa-0.9.1-2196F3?logo=python&logoColor=white)](https://pypi.org/project/librosa/0.9.1/)
[![pytorch-lightning](https://img.shields.io/badge/pytorch--lightning-1.9.5-792EE5?logo=lightning&logoColor=white)](https://pypi.org/project/pytorch-lightning/1.9.5/)

```bash
pip install speechbrain==1.0.3
pip install librosa==0.9.1
pip install pytorch-lightning==1.9.5
pip install -r requirements.txt
```

### 2. Download the MLAAD v8 Dataset and Protocols

Our experiments are conducted on the MLAAD benchmark. You can access the dataset and our newly proposed evaluation protocols via the links below:

* **Dataset:** [MLAAD](https://huggingface.co/datasets/mueller91/MLAAD) 
* **Proposed Protocols:** [MLAAD Protocols for Source-Speaker Disentanglement](https://huggingface.co/datasets/xxuan-speech/MLAAD_Protocols_for_Source-Speaker_Disentanglement_Research)

### 3 Train

```bash
python main.py
```

### 4 Test

```bash
python test.py
```

### 5. Inference on a pair of .wav files

You can use demo/demo.ipynb to test a pair of .wav files by replacing their paths and performing inference to estimate whether the audios were generated by the same or different synthesis sources.
